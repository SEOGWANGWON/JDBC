SelectOne
	selectOne : 주로 단일 행 (레코드) 검색하는 데 사용
				일반적으로 WHERE절을 사용해서 조건을 지정하고
				결과로 단일 객체나 레코드를 반환
				결과가 없거나 여러 결과가 나오는 경우
				첫번째 결과만 주로 반환
				주요키(Primary key) 값으로 검색하거나
				고유한 결과를 필요로 하는 경우에 사용
